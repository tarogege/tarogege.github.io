专有名词

  • 数组每个维度叫做axis 轴
  • axes的数量叫做rank
    比如 3*4的数组，叫做rank2，第一个轴长度为3，第二个轴长度为4
  • 轴长度list被称为shape （3，4）
    size = shape所有元素的乘积
